## Estructura de un HTML

- Body
  - Header (Encabezado)
    - Nav
  - Section (Parte izquierda)
    - Article
  - Aside (Parte derecha)
  - Footer

Hay que utilizar las etiquetas correctas ya que ayuda al posicionamiento web en los navegadores.
